# IRIS

Install the project:

```sh
$ npm install
```

Available commands:

```sh
# run the dev server
$ npm run dev

# run test suite
$ npm run test

# run linting
$ npm run lint

# build for production
$ npm run build

# run the prod server
$ npm start
```

The following pre-commit hooks are enabled:

- `lint:staged` - lint staged JS files against [semistandard](https://github.com/Flet/semistandard)
